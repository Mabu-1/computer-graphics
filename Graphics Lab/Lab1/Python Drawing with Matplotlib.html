python
import matplotlib.pyplot as plt

  1. Drawing a Circle:
python
import matplotlib.pyplot as plt
from matplotlib.patches import Circle

# Create a figure and axis
fig, ax = plt.subplots()

# Create a circle with center (2, 2) and radius 1
circle = Circle((2, 2), 1, fill=False, color='blue')

# Add the circle to the axis
ax.add_patch(circle)

# Set axis limits to fit the circle
ax.set_xlim(0, 4)
ax.set_ylim(0, 4)

# Show the plot
plt.gca().set_aspect('equal', adjustable='box')
plt.show()
  1. Drawing a Rectangle:
python
import matplotlib.pyplot as plt
from matplotlib.patches import Rectangle

# Create a figure and axis
fig, ax = plt.subplots()

# Create a rectangle with lower-left corner at (1, 1), width 3, and height 2
rectangle = Rectangle((1, 1), 3, 2, fill=False, color='red')

# Add the rectangle to the axis
ax.add_patch(rectangle)

# Set axis limits to fit the rectangle
ax.set_xlim(0, 5)
ax.set_ylim(0, 5)

# Show the plot
plt.gca().set_aspect('equal', adjustable='box')
plt.show()

These examples demonstrate how to draw a line, circle, and rectangle using Python's matplotlib library. You can customize the coordinates, colors, and other properties according to your needs.

Certainly! You can use the Python library turtle to draw a line, circle, and rectangle. The turtle library provides a simple way to create drawings using a turtle that moves on the screen. Here are examples for drawing a line, circle, and rectangle using turtle:

  1. Drawing a Line:
python
import turtle

# Create a turtle object
t = turtle.Turtle()

# Move the turtle to the starting position
t.penup()  # Lift the pen to move without drawing
t.goto(-50, 0)  # Set the starting coordinates
t.pendown()  # Lower the pen to start drawing

# Draw a line
t.forward(100)  # Move the turtle forward by 100 units

# Close the turtle graphics window when clicked
turtle.exitonclick()
  1. Drawing a Circle:
python
import turtle

# Create a turtle object
t = turtle.Turtle()

# Draw a circle with radius 50
t.circle(50)

# Close the turtle graphics window when clicked
turtle.exitonclick()
  1. Drawing a Rectangle:
python
import turtle

# Create a turtle object
t = turtle.Turtle()

# Draw a rectangle
for _ in range(2):  # Repeat the following instructions twice to draw a rectangle
    t.forward(100)  # Move forward by 100 units
    t.left(90)  # Turn left by 90 degrees
    t.forward(50)  # Move forward by 50 units
    t.left(90)  # Turn left by 90 degrees

# Close the turtle graphics window when clicked
turtle.exitonclick()

The t.circle(radius, extent) method in the turtle library is used to draw a portion of a circle with a specified radius and extent (angle).

In the example t.circle(-30, 140), here's what each parameter means:

  • radius: This is the radius of the circle. If the radius is positive, it will draw the circle counterclockwise, and if it's negative, it will draw the circle clockwise.

  • extent: This is the angle in degrees that determines how much of the circle to draw. In this case, it's 140 degrees, which means it will draw 140 degrees of a circle.

So, t.circle(-30, 140) is drawing a clockwise arc with a radius of 30 units, covering an angle of 140 degrees. This is often used to create curved shapes or arcs in turtle graphics.

In the turtle graphics library in Python, the t.setheading(angle) function is used to set the heading or direction of the turtle, where the angle argument determines the new direction.

  • When you use t.setheading(0), it points the turtle to the right (east).
  • When you use t.setheading(90), it points the turtle upward (north).
  • When you use t.setheading(180), it points the turtle to the left (west).
  • When you use t.setheading(270), it points the turtle downward (south).

In your code, t.setheading(120) sets the turtle's heading to 120 degrees, which is between pointing upwards and pointing to the left (northwest direction).
